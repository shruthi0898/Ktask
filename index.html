Ruby on Rails:

Description: Learning to build a modern web application is strenuous, and if you are not a coder, it’s nigh impossible. All that changes when you use Ruby on Rails. This framework makes the process fun and easy. Learn how to use it, right from scripting to deployment at our Ruby on Rails workshop and show off your technical prowess! If you see yourself building the next Github, Hulu, Slideshare, Twitter or any of the hundreds of the most popular sites today taking advantage of this framework, then don’t miss out on this opportunity! The workshop is for two days.This Workshop is not constrained to any departments.

Augmented Reality and Virtual Reality - Build your own Pokemon Go

Description: Wouldn’t it be absolute fun to actually run in instead of play with Temple Run? How exciting do you think it would be if you could create one such application all by yourself? No matter how many games we play, virtual games never fail to entice us. Here we are, to teach you the skills required to turn this magic into reality. The Augmented and Virtual Reality workshop is where you start learning right from the basics of AR and VR and move on all the way to advanced concepts till you are equipped with the skill to develop your own apps and experience them using the Google Gears. The workshop is for two days. This Workshop is not constrained to any department.

XCEED IN SALEM,2016

Kick-starting Kurukshetra's activities for the year, the Xceed team set the count rolling with the following workshops conducted at Annaporna College of Engineering, Salem.
  1. Ruby on Rails - Tweet it like Twitter
  2. Augmented and Virtual Reality - Build your own Pokemon Go
With the participants gaining momentum and getting inquisitive, the output was stellar with lots of positives to take away. Here's a pictorial sneak peek into the proceedings of Xceed.
